markdown教程

发表于 2017-08-29

标题

  首先是标题啦，标题有两种写法，下划线模式和井号模式，下划线模式只有两种大小不太实用，所以这里我就只说说井号模式，一共有六种大小，从一个#(最大)到六个#(最小)，注意#和标题之间要有空格间隙，不然不会被视作标题

# 标题1
## 标题2
### 标题3

当然也可以同时在两边加#，这样看起来更好看一点

# 标题 #

引用区块

  你应该在一些公众号那里看过这种效果，类似于

这里是一个引用

这样的写法很简单，只要在开头加上一个>就可以，也要注意>和正文之间需要有一个空格

> 这里是一个引用

列表

  markdown支持有序列表和无序列表，无序列表可以用星号，加号，减号作为列表标记

* red
* green
* ...

或者

- red
- green
- ...

而有序列表只需要换成数字就行

1. a
2. b
3. c

分割线

  分割线可以这样写

* * *
***
*****
- - -
------------------

上面的语法都是对的

链接

[链接上的文字](链接的地址 "Title(可选)")

效果就是这样：链接上的文字

强调

使用两个*夹起来的文字会作为强调格式，而用四个*夹起来的文字会变成strong格式,注意这里就没有空格了

*强调*
**strong**

代码

一块代码只需要用一个tab缩进就可以了，在行间的代码可以用反引号`括起来

图片

其实这个网站插图片比较麻烦，不过也可以插入，你只要这样写
![](图片文件名.后缀名)
然后把图片发给我就好啦

缩进

缩进有自己的代码
&emsp; 表示缩进
